# Beanvest

## Supported JDK
Java 17 is minimum required. Application can be built to native binaries using GraalVM. It can be downloaded from official website: https://www.graalvm.org/downloads/#

## Running tests
```bash
./gradlew test
```

## Building binaries
```bash
./gradlew nativeCompile
```
